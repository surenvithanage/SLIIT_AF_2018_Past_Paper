##This is the sample react application for starting react and express based projects.

This includes;

    express
    mongoose
    react
    react-dom
    react-router-dom
    bootstart
    parcel
 
 **Install dependencies**
 
    npm intall
 
 **Start**
 
 Make sure mongo DB is running on localhost 27017.
 
    npm start
